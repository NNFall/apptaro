# Old Design Archive

This folder stores previous UI versions that are no longer active in the main app build.

## Current archive

- `flutter_ui_v1/` — the original dashboard/tab-based Flutter client before the Telegram-style redesign.

## Why this exists

- to preserve the first working mobile/web UI iteration;
- to allow rollback or selective reuse of widgets and flows later;
- to keep design experiments separate from the active client.
