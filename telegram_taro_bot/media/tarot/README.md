# Tarot Media

Place assets here:

- `cards/` - tarot card images (`.png/.jpg/.jpeg/.webp`)
- `backgrounds/main.png` - spread background image
- `layout.json` - card positions and angles for a 3-card spread

Card file name is used as card title:

- `wheel_of_fortune.png` -> `Wheel Of Fortune`
- `–Ї–Њ–ї–µ—Б–Њ_—Д–Њ—А—В—Г–љ—Л.jpg` -> `–Ъ–Њ–ї–µ—Б–Њ –§–Њ—А—В—Г–љ—Л`
